<h2>AJAX</h2>
AJAX就是异步 JavaScript 和 XML，是一种用于创建快速动态网页的技术。                                
通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更
新。从而减少服务的负荷压力。                              
传统的网页（不使用AJAX的情况下）如果需要更新内容，必需重载整个网页面，从而延长了加载的速度，这是硬伤。                        
AJAX 基于已有的标准。这些标准已被开发者使用多年。
